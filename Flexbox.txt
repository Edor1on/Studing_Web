Супер, Flexbox Froggy — це база! Якщо пройшов сам, значить логіку осей ти вже зрозумів.

Ось твоя шпаргалка ("чіт-шіт"), розділена на властивості для **Батька (Container)** і для **Дітей (Items)**.

### 1. Для Батька (Container)

`display: flex;` — вмикає флекс-режим.



#### `flex-direction` (Куди йде вісь?)
row — зліва направо (стандарт)
row-reverse — справа наліво
column — зверху вниз
column-reverse — знизу вверх

#### `justify-content` (Вирівнювання по ГОЛОВНІЙ осі — зазвичай горизонтально)
flex-start — притиснути до початку
flex-end — притиснути до кінця
center — по центру
space-between — крайнi по боках, проміжки рівні
space-around — відступи навколо кожного елемента
space-evenly — абсолютно рівні відступи всюди

#### `align-items` (Вирівнювання по ПОПЕРЕЧНІЙ осі — зазвичай вертикально)
stretch — розтягнути на всю висоту (стандарт)
flex-start — притиснути до верху
flex-end — притиснути до низу
center — по центру поперек
baseline — по лінії тексту (щоб букви стояли рівно)

#### `flex-wrap` (Перенос рядків)
nowrap — в один рядок (стискає елементи)
wrap — переносити на новий рядок
wrap-reverse — переносити (але рядок з'являється зверху)

#### `align-content` (Якщо є КІЛЬКА рядків — як `wrap`)
Працює як justify-content, але для цілих рядків по вертикалі*
flex-start — всі рядки зсунуті до початку
center — всі рядки по центру
space-between — рядки розкидані по краях

---

 2. Для Дітей (Items)

`align-self` (Перевизначає `align-items` для одного елемента)
auto — як у батька
flex-start / center / flex-end — індивідуальне вирівнювання

`flex-grow` (Жадібність / Розширення)
0 — не збільшуватись (стандарт)
1 — займати весь вільний простір
(Можна ставити 2, 3... це пропорції)*

`flex-shrink` (Стиснення)
1 — стискатись, якщо мало місця (стандарт)
0 — не стискатись (тримати розмір до останнього)

**`order`(Порядок)
0 — на своєму місці
-1 — перемістити раніше
1 — перемістити пізніше

**Короткий запис:
`flex: 0 1 auto;` — це скорочення для `grow`, `shrink`, `basis`.
(Не рости, стискайся, розмір авто).
